OptoPrimDB

OptoPrimDB is an early-stage, experimental database aiming to collect and organize published optogenetics studies in primates, including humans. It provides a searchable overview of the literature using standardized, paper-level metadata.

This is a very early testing version. The database is incomplete and the structure is still evolving.
